## 📌 Sobre o projeto

O **AquaCicle** é um sistema inteligente de monitoramento e reaproveitamento de água utilizando **Arduino** e **ESP32**.

O projeto automatiza o controle e monitoramento da água da chuva, garantindo melhor uso de recursos hídricos através de sensores e tecnologias IoT.

---

## 🚀 Tecnologias utilizadas

- Arduino  
- ESP32  
- C++  
- IoT (Internet das Coisas)  
- Sensores de pH  
- Sensor de turbidez  
- Sensores de nível  
- Web Server (ESP32)  

---

## ⚙️ Como funciona

### 🔹 Arduino
Responsável por:
- Monitorar sensores de nível  
- Controlar bombas de água  

### 🔹 ESP32
Responsável por:
- Ler sensores de pH e turbidez  
- Processar os dados  
- Exibir informações via navegador
